This is a basic Java 2D game.
You have 4 levels, and you finish the game either by losing or by killing all the objects
in all levels.

If you hit an asteroid, an enemy's ship or an enemy's bullet, you will lose a life.

Basic commands:
	- Up: move up
	- Down: move down
	- Left: rotate left
	- Right: rotate right
	- Space: shoot
